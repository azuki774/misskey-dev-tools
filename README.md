# misskey-dev-tools


### count-reaction send
- 過去100回のリアクションからトップ5に使われたリアクションをまとめてノートします
    ```
    USERID="********" I="************" DOMAIN="misskey.io" build/bin/count-reaction send
    ```

    - 出てくるノート
        ```
        最近よく使うリアクションランキング
        :ablobcat_headbangfastultra: x10
        :ablobcat_cryingcute: x6
        :kusa: x6
        :ablobcat_resonyance: x6
        :neko_jii_nya2: x3
        （過去 100回のリアクションから）
        ```
